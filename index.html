🚫 Coin won't work when opened as a local file

You're opening this as a file:// URL (double-click). Chrome blocks microphone and tab audio capture on local files — transcription will silently fail.

How to fix — pick one:
A
Easiest — use VS Code Live Server: Install the Live Server extension, right-click coin.html → Open with Live Server. Opens at http://localhost:5500
B
Python (if installed): In Terminal, cd to the folder containing coin.html, run python3 -m http.server 8080, then open http://localhost:8080/coin.html
C
Node (if installed): Run npx serve . in the folder, open the URL it gives you.
